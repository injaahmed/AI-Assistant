# Super AI Assistant

A powerful local AI assistant built using Python, Streamlit, Ollama, and Phi3.

## Features

- Normal AI Chat
- Coding Assistant
- PDF Reader AI
- Internet Search AI
- Voice AI
- Chat Memory
- ChatGPT-style UI

## Technologies Used

- Python
- Streamlit
- Ollama
- Phi3
- PyPDF2
- DuckDuckGo Search

## How To Run

### Install Libraries

pip install -r requirements.txt

### Run Ollama

ollama run phi3

### Run AI Web App

python -m streamlit run app.py
